### Ongoing Development
Nothing significant is planned between now and next year. However, any tweaks, refinements, or issues I address in Xprime will be part of Xprime 27, codenamed Opux — inspired by the Latin word Opus, meaning “work” or “creation,”.

- The program or application icon is now displayed in the Window menu.

- Fixed an issue in Project Settings where the Advanced window was resizable when it should not have been.


### Xprime 26.5.x
The 26.5 update has been refinement, resolving minor issues, and bringing everything closer to final completion.

In light of HP Prime Development Tools, Xprime adopted the .hpppl and .hppplplus file types and discontinue .ppl .ppl+ .prgm+. This change simplifies development for those using either Visual Studio Code or Xprime.

>[!NOTE]
>Xprime also supports .hpppl+ as well as .hppplplus; however, due to the + character potentially causing issues on some systems and websites, using .hppplplus is recommended.
